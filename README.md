# Captions (captions-ai)

Captions is an AI-powered video creation and editing platform developed by Mirage, a New York-based AI research and product company with over 20 million users. The platform enables creators and businesses to generate, caption, dub, translate, and publish talking-head videos at scale using generative AI. The Captions API — served under the Mirage API umbrella — provides programmatic access to video captioning, AI Creator video generation using community avatars or AI Twin digital clones, AI Ads video generation, and bulk video management. All API operations use API key authentication via the `x-api-key` header and are billed on a per-second-of-video credit model or per-minute of input video.
